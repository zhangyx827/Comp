# 编译器设计与实现报告 (Compiler Design & Implementation Report)

## 一、项目概述
本项目实现了一个**自定义编程语言（类简单C语言）**的完整编译器。不仅完成了基本的分组作业要求（词法分析、语法分析、语义分析），并且**完全自己实现了词法、语法分析程序**（这属于作业要求的**重点加分项目**），没有依赖外部的 Flex/Bison 或 `.y` 文件生成器。在此之上，项目还进一步实现了**代码优化**、**x86汇编代码生成**，并提供了一个美观强大的**Web可视化平台**展示编译的每一个阶段。

## 二、项目结构
在经过功能完善和整理后，项目呈现非常清晰的模块化架构：

- `compiler/`：编译器核心代码目录。
  - `lexer.py`：词法分析器，完全自实现的正则表达式状态机。
  - `parser.py`：语法分析器，完全自实现的递归下降解析器，直接生成抽象语法树(AST)。
  - `semantic.py`：语义分析器，通过遍历语法树实现语法制导翻译，完成类型检查和符号表管理。
  - `optimizer.py`：代码优化器，实现了简单的常量折叠等优化。
  - `codegen.py`：代码生成器，输出Intel格式的x86汇编代码。
  - `compiler.py`：串联所有步骤的主控类。
  - `core.py`：定义 Token、ASTNode等核心数据结构。
- `web/`：可视化前端与后端服务。
  - `app.py`：Flask服务器API接口。
  - `templates/` & `static/`：现代化的Web UI前端界面。
- `examples/`：存放了多种测试示例代码（如循环、函数、基本运算）。
- `main.py`：项目启动入口。

## 三、功能特性详解

### 1. 词法分析 (Lexical Analysis)
实现了自定义语言的词法分析，支持识别各种关键字（`int`, `float`, `while`, `for`, `if`, `return` 等），标识符，字面量（整数、浮点、字符串、布尔值），以及丰富的运算符符号。

### 2. 语法分析 (Syntax Analysis)
采用**自顶向下的递归下降分析法**实现。能够处理变量声明、函数声明、复杂的控制流（if-else, while, for）以及具有优先级规则的表达式，最终构建出完整的抽象语法树（AST）。

### 3. 语义分析 (Semantic Analysis)
**完全替代了传统的`.y`文件方式**，通过遍历AST节点，实现了语法制导的翻译过程：
- **符号表管理**：记录变量类型、函数签名。
- **作用域检查**：控制局部变量和全局变量的作用域。
- **类型检查**：对运算操作数类型、函数返回值类型等进行严格校验。

### 4. 汇编代码生成与优化 (Code Generation)
支持初步的常量折叠优化，并将AST转换为可执行的**x86汇编指令**（支持局部变量在栈上的偏移计算和控制流跳转）。

### 5. Web可视化平台 (Visualization)
- **语法高亮编辑器**：支持多种内置代码模板快速切换（基础运算、循环、函数等）。
- **流程可视化**：直观展示编译过程的每一个步骤（词法Token流、树形AST结构、符号表数据、生成的汇编代码）。
- **错误捕获**：在界面中直接高亮显示语法和语义错误。
